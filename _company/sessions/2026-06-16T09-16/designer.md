# 📊 잠재 수익 분포 시각화 프로토타입: 최종 디자인 스펙 (Figma 준비용)

## 1. 시각화 목표 및 핵심 메시지

*   **최종 목표:** 데이터의 복잡성(분포)을 단순화하여, **'신뢰도 기반의 효율적인 수익 경로'**라는 단일하고 강력한 메시지를 전달한다.
*   **핵심 메시지:** "데이터 신뢰도($\lambda$)를 기반으로 예측된 잠재 수익 분포를 명확히 제시한다."

## 2. 레이아웃 구조 (Layout Structure)

**레이아웃 타입:** 대시보드/인포그래픽 혼합형 (Dashboard/Infographic Hybrid)
**전체 컨셉:** 좌측에 신뢰도(Trust) 요소를 배치하고, 중앙에 핵심 분포 시각화, 우측에 실행 가능한 인사이트를 배치하여 흐름을 유도한다.

| 영역 | 구성 요소 | 목적 및 내용 | 디자인 강조점 |
| :--- | :--- | :--- | :--- |
| **A. 헤더 (Top Bar)** | 타이틀, 핵심 KPI 요약 | 시각화의 주제와 가장 중요한 결과($Margin_{Weighted}$)를 즉시 제시. | 브랜드 컬러 사용, 명확하고 간결하게. |
| **B. 신뢰도 섹션 (Trust Section - 좌측)** | 데이터 무결성 지표 ($\lambda$) | 시스템의 신뢰도를 시각적으로 강조하여 데이터의 근거를 확보. | $\lambda$ 값을 게이지, 또는 별도의 색상 톤으로 표현하여 '안정성'을 어필. |
| **C. 핵심 분포 (Core Visualization - 중앙)** | 잠재 수익 분포 그래프 (2D/3D) | X축 ($\text{Demand Index}$)과 Y축 ($\text{Profit Index}$) 간의 관계를 명확히 보여줌. | 가장 큰 시각적 영역을 할애하며, 데이터 밀도를 표현하는 데 집중. |
| **D. 실행 인사이트 (Actionable Insights - 우측)** | 수익화 모델 비교 및 권고안 | 분포 결과에 따른 구체적인 비즈니스 액션(모델 B 추천 근거) 제시. | 명확한 텍스트 박스와 아이콘을 사용하여 '왜' 이 결과가 중요한지 설명. |

## 3. 핵심 컴포넌트 사양 (Core Component Specifications)

### 3.1 그래프 상세 사양 (C 섹션: Core Visualization)

*   **그래프 타입:** **2D 히트맵/밀도 플롯 (Heatmap / Density Plot)**을 기반으로 한 영역 채우기 시각화가 가장 강력함.
*   **X축 ($\text{Demand Index}$):** 0부터 100까지의 연속적인 스케일로 설정. (좌측: Low Demand, 우측: High Demand)
*   **Y축 ($\text{Profit Index}$):** 최소 마진($\lambda$ 기반 최소값)부터 최대 마진까지의 연속적인 스케일.
*   **데이터 표현:** 각 $(\text{Demand}, \text{Profit})$ 좌표에 **색상 농도(Density)**를 적용하여 잠재 수익 분포의 밀도를 표현한다. (고밀도는 진한 색, 저밀도는 옅은 색)
*   **레이어 오버레이:** 확정된 **수익화 모델 B**의 경계선 또는 최적 영역을 반투명한 오버레이 레이어로 추가하여 시각적 가이드라인을 제공한다.

### 3.2 데이터 신뢰도 강조 컴포넌트 (B 섹션)

*   **컴포넌트:** $\lambda$ 지표 게이지/스코어 카드
*   **디자인:** 원형 또는 바 형태의 **신뢰도 게이지(Trust Gauge)**를 사용하여 현재 시스템 무결성($\lambda$) 점수를 표현한다.
    *   **색상 매핑:** $\lambda$ 값이 높을수록 (예: 0.9 이상) 녹색 계열, 중간값은 노란색, 낮을수록 주황/빨간색으로 변화시켜 직관적으로 신뢰도를 전달한다.
*   **캡션:** "시스템 무결성 ($\lambda$): [실제 $\lambda$ 값] / 기준치: [기준치]" 명시.

### 3.3 실행 권고 컴포넌트 (D 섹션)

*   **컴포넌트:** '최적 경로' 박스 (Recommendation Box)
*   **내용:** 가장 신뢰도 높은 모델 B를 **Primary Recommendation**으로 강조한다.
    *   **헤드라인:** "**권장 전략: 수익화 모델 B (번들 옵션)**"
    *   **근거 요약:** "신뢰도 $\lambda$와 $Margin_{Weighted}$ 분석 결과, 번들 옵션이 가장 높은 안정적 수익 경로를 제공합니다."
*   **액션 버튼:** '전체 보고서 다운로드' 또는 '가격 전략 상세 보기' 등 명확한 CTA(Call to Action) 버튼을 배치한다.
